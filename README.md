# Spdx3

A NuGet library (.NET 8) for creating, reading, and
writing [Software Bills of Materials](https://www.ntia.gov/page/software-bill-materials) files
in [SPDX 3](https://spdx.github.io/spdx-spec/v3.0.1/) format.
